# ComputerVisionDemo

Demosntração de rede convolucional simples para classificação de digitos da base de dados MNIST no navegador.
Biblioteca de redes neurais em JS por Andrej Karpathy.

# Uso

No terminal, abra um servidor utilizando Python:
 ```bash
.../ComputerVisionDemo $ python -m http.server <porta>
 ```

 No navegador abra o enderenço `localhost:<porta>`


# Estrutura de arquivos

```
.                                                         
├── build       # Biblioteca         
├── data		# Dados utilizados
├── src			
└── index.html
```
